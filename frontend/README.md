# Garden Shop

Это онлайн-магазин садовых товаров, созданный с помощью React. Сайт позволяет пользователям просматривать садовые товары по категориям, смотреть детали продуктов и видеть специальные предложения и скидки.

## Возможности

- Просмотр товаров по категориям
- Просмотр деталей товаров с изображениями и ценами
- Специальный раздел скидок с товарами со скидкой
- Адаптивный дизайн для мобильных и десктопных устройств
- Регистрация пользователей и предложения скидок

## Используемые технологии

- React 19
- Redux Toolkit для управления состоянием
- React Router для навигации
- Express.js бэкенд с SQLite базой данных
- Sequelize ORM

## Начало работы

### Предварительные требования

Убедитесь, что у вас установлен Node.js на системе.

### Установка

1. Клонируйте репозиторий
2. Установите зависимости фронтенда:
   ```bash
   cd frontend
   npm install
   ```

3. Установите зависимости бэкенда:
   ```bash
   cd backend
   npm install
   ```

### Запуск приложения

1. Запустите сервер бэкенда:
   ```bash
   cd backend
   npm run dev
   ```
   Бэкенд будет работать на http://localhost:3333

2. Запустите фронтенд:
   ```bash
   cd frontend
   npm start
   ```
   Фронтенд будет работать на http://localhost:3000

### Альтернативный порт для фронтенда (если 3000 занят):
```bash
cd frontend
PORT=3001 npm start
```

## Доступные скрипты

В директории фронтенд проекта вы можете запустить:

- `npm start` - Запускает приложение в режиме разработки
- `npm test` - Запускает тестовый раннер
- `npm run build` - Собирает приложение для продакшена
- `npm run eject` - Извлекает из Create React App (необратимо)

## Структура проекта

- `src/components/` - Переиспользуемые UI компоненты
- `src/pages/` - Компоненты страниц
- `src/store/` - Redux store и API срезы
- `public/` - Статические ресурсы

## Бэкенд

Бэкенд находится в папке `backend` и предоставляет REST API эндпоинты для:
- Управления категориями
- Управления товарами
- Обработки заказов
- Скидок/распродаж
